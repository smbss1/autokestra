# v0.1 Quickstart (local)

## Prerequisites

- Bun (recommended for development)

## 1) Install dependencies

From the repo root:

- `bun install`

## 2) Start the server

### Option A — run the CLI directly (no install)

- `bun packages/cli/src/cli.ts server start -c ./config.example.yaml`

### Option B — install a CLI command (`workflow` / `autokestra` / `cli`)

From the repo root:

- `cd packages/cli`
- `bun link`

Then:

- `workflow server start -c ./config.example.yaml`

## 3) Apply an example workflow via API

An example workflow is available at:
- `examples/workflows/v0.1-happy-path.yaml`

Apply it via the server API (requires API key configured in `config.example.yaml`):

- `curl -X PUT "http://localhost:7233/api/v1/workflows/v01-happy" \
  -H "Authorization: Bearer <your-api-key>" \
  -H "Content-Type: text/yaml" \
  --data-binary @examples/workflows/v0.1-happy-path.yaml`

List workflows:

- `curl -H "Authorization: Bearer <your-api-key>" "http://localhost:7233/api/v1/workflows"`
